ProXperty 

ProXperty adalah platform manajemen penyewaan properti yang dirancang untuk menciptakan transparansi dan ketertiban antara pengelola (pemilik) dan penyewa. Sistem ini fokus pada pemantauan status sewa, pengelolaan data properti, serta transparansi tagihan secara terstruktur.

---

## Fitur Utama

Project ini mencakup berbagai kebutuhan fungsional untuk memudahkan ekosistem penyewaan:
1. Manajemen Properti: Tambah, ubah, dan hapus data properti secara real-time.
2. Pencarian Cerdas: Filter properti berdasarkan kriteria tertentu (lokasi, harga, dll).
3. Sistem Transaksi & Tagihan: Kalkulasi otomatis untuk masa sewa dan pemantauan status pembayaran.
4. Komunikasi Terintegrasi: Fitur pesan langsung antara penyewa dan pemilik.
5. Panel Admin: Monitoring seluruh transaksi dan pengelolaan data pengguna dalam satu dashboard.

## Peran Pengguna (User Roles)

1.  Penyewa: Mencari properti, melakukan permintaan sewa, dan memantau tagihan.
2.  Pemilik Properti: Mengelola listing properti dan memproses permintaan sewa.
3.  Admin: Mengawasi jalannya sistem dan memastikan keamanan transaksi.

## Tech Stack

Sistem ini dibangun menggunakan teknologi yang menjamin stabilitas dan kemudahan pemeliharaan:
1. Languages: PHP (Backend), HTML/CSS (Frontend)
2. Database: MySQL
3. Server: Apache (XAMPP/Environment serupa)
4. Tools: VSCode, MySQL Workbench, Jira (Project Management)
